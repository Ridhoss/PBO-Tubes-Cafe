# APLIKASI MANAJEMEN CAFE
Aplikasi ini merupakan aplikasi manajemen cafe sederhana yang dibuat menggunakan bahasa pemrograman Java. Aplikasi ini memungkinkan pengguna untuk mengelola menu, pesanan cafe dengan mudah.

## Cara melakukan compile aplikasi

1. Clone repository ini ke dalam komputer/laptop Anda.
```bash
git clone https://github.com/Ridhoss/PBO-Tubes-Cafe.git
```

2. Instal Java Development Kit (JDK) versi 11 atau lebih tinggi.

3. Install Netbeans IDE sebagai code editor (Java) jika ingin menggunakan Netbeans.

4. Buka project di Netbeans IDE.

5. Jalankan Main.java yang berada di package main.
